== Description ==

O VitaPro Clinic é um tema WordPress Full Site Editing (FSE) completo e profissional, especialmente projetado para clínicas médicas, consultórios, hospitais e profissionais de saúde em geral.

= Características Principais =

* **Full Site Editing (FSE)**: Totalmente construído com blocos do WordPress
* **Design Profissional**: Interface limpa, moderna e confiável
* **Responsivo**: Otimizado para todos os dispositivos (mobile-first)
* **Acessibilidade**: Desenvolvido seguindo as melhores práticas de acessibilidade
* **Performance**: Código otimizado para velocidade e SEO
* **Custom Post Types**: Serviços, Profissionais, Depoimentos e Convênios
* **Padrões de Blocos**: Seções pré-construídas para facilitar a criação de conteúdo
* **Múltiplos Templates**: Templates personalizados para diferentes tipos de página

= Custom Post Types Incluídos =

* **Serviços/Tratamentos**: Para gerenciar os serviços oferecidos pela clínica
* **Profissionais/Equipe**: Para apresentar a equipe médica
* **Depoimentos**: Para exibir depoimentos de pacientes
* **Convênios**: Para listar os planos de saúde aceitos

= Padrões de Blocos Incluídos =

* Seção Hero para homepage
* Grid de serviços
* Grid de equipe médica
* Seção de depoimentos
* Formulário de contato
* Seções de call-to-action
* E muito mais...

= Paleta de Cores =

* Azul Primário (#3A7DBC) - Transmite confiança e profissionalismo
* Verde Secundário (#64C5BA) - Representa saúde e tranquilidade
* Tons neutros para perfeita harmonia visual

= Tipografia =

* Fontes: Inter e Manrope
* Otimizada para legibilidade
* Tamanhos responsivos (fluid typography)

= Instalação =

1. Faça o upload do tema via Painel do WordPress ou FTP
2. Ative o tema em Aparência > Temas
3. Acesse Aparência > Editor de Site para começar a personalizar
4. Use os padrões de blocos incluídos para criar suas páginas rapidamente

= Personalização =

Este tema foi projetado para ser facilmente personalizável através do Editor de Site do WordPress. Você pode:

* Modificar cores, tipografia e espaçamentos via theme.json
* Criar novos padrões de blocos
* Customizar templates e partes de template
* Ajustar layouts sem conhecimento de código

== Installation ==

1. No painel do WordPress, vá para Aparência > Temas
2. Clique em "Adicionar Novo"
3. Clique em "Enviar Tema"
4. Selecione o arquivo ZIP do tema
5. Clique em "Instalar Agora"
6. Após a instalação, clique em "Ativar"

== Frequently Asked Questions ==

= Este tema é responsivo? =

Sim, o VitaPro Clinic é totalmente responsivo e foi desenvolvido com a abordagem mobile-first.

= Posso personalizar as cores? =

Sim, todas as cores podem ser facilmente personalizadas através do Editor de Site do WordPress ou editando o arquivo theme.json.

= Como adiciono novos serviços? =

Vá para Serviços > Adicionar Novo no painel administrativo. O tema inclui campos personalizados para informações detalhadas sobre cada serviço.

= O tema é compatível com plugins? =

Sim, o tema foi desenvolvido seguindo os padrões do WordPress e é compatível com a maioria dos plugins populares.

= Posso usar este tema para outros tipos de site? =

Embora tenha sido desenvolvido especificamente para clínicas médicas, você pode adaptá-lo para outros tipos de negócios relacionados à saúde e bem-estar.
